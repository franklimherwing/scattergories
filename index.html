How to Play

  • Click "Start Game" to begin a 3-minute round with a random letter
  • Fill in each category with a word that starts with the given letter
  • When time is up, score points for each unique answer
  • Use "New Letter" to start a new round with a different letter
  • Score 1 point for each valid answer, 2 points if no other players have the same answer

Note: This is a single-player version. For multiplayer, players would compare answers after each round.
